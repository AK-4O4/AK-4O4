## 🚀 Current Projects

### 🎵 Music Player — Flutter
A cross-platform mobile music player built with Flutter. Focused on clean UI, smooth playback controls, and a great user experience on both Android and iOS.

### 🐍 CV-Based Snake Game — React
A twist on the classic Snake game where computer vision drives the gameplay. Built with React, using real-time input processing to control the snake — because why use a keyboard when you can use yourself?

### 🖼️ Image Steganography App — React + FastAPI
A full-stack application that hides and reveals secret messages within images using steganography techniques. React frontend communicates with a FastAPI backend to handle encoding and decoding seamlessly.
